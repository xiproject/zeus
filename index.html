Introduction

Zeus is your personal assistant. He answers your questions, plays music, helps you get stuff done, and much more. He runs on your computer, understanding your behaviour and adapting to it. Best of all, he's free-as-in-speech, and designed to be extended and modified to suit your needs - like a hacker's assistant should be.

Installation

(Tested using node v0.10)

$ git clone https://github.com/xiproject/zeus # Get the code
$ ./scripts/update.sh # Fetch git submodules and npm modules
$ ./scripts/run.sh # Good to go!

If you'd like to pretty print the error logs, get bunyan. For example,

$ ./scripts/run.sh | bunyan

Input

For speech-to-text, install the Chrome app. This uses Google speech-to-text. The CLI is very handy too.

$ ./scripts/cli.sh

Things you should know

As your personal assistant, Zeus is privy to a lot of information about you. You should understand what happens to this data and the security implications of using Zeus.

  • Zeus is still in development, and is rough around the edges. It has no security. Don't run it on a network you don't trust.
  • Zeus accesses your camera and microphone to get information about you and your surroundings. The camera feed is never sent to any other server. The microphone feed is sent to Google for speech-to-text through the Chrome app.
