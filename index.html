Zeus is a personal assistant for hackers.


Meet Zeus




Distributed

Zeus is a network of agents. These agents can run on any device - your laptop, your phone, even your toaster - and talk to each other using a powerful HTTP API.



Batteries Included

Zeus builds on the powerful capabilites of existing open source software and free-to-use web services. Zeus can play music, answer your questions, check email, and more.



Open Source and Fully Extensible

Zeus is powered by the Xi framework. You can customize Zeus by writing your own Xi agent in Javascript, or tinker with the existing agents if you prefer - it's all GPLv3.


Get started


$ git clone https://github.com/xiproject/zeus && cd zeus
$ scripts/install.sh
$ scripts/run.sh
You will probably run into problems, so RTFD.
